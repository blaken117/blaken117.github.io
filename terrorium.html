Terrorium

2D Aerial Bullet-Hell

Terrorium was a semester long Uni project which covered all aspects of game creation including; ideation, documentation, development, play-testing, publishing and marketing. The aim of the game was to use the same mechanics from the game luftrausers however add our own spin into the game.

Contributions

  • Coded AI Targeting systems
  • Setup aerial controls using physics and drag calculations
  • Scripted scoring and viewership features
  • Designed 2D environment
  • Setup bug customization via Player Prefs and JSON
