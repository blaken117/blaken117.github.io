Dis Structure

Optical Art Game

Dis Structure was a 3-week Uni project which required me to create a game based on an assigned artist. The aim was to communicate the meaning and purpose of their art via gameplay mechanics. I decided to give the player the ability to create their own op-art, as op-art can have different meanings to other people. Allowing this freedom of expression related heavily to Bridget Riley's process in creating her art.

Contributions

  • Created 2D art and UI elements via Adobe Illustrator
  • Built the game via Web GL (playable via browser)
  • Setup screen capture for saving of created art
  • Coded grid system for AI/ Player movement and effects

    • Proficiency Gained

    Unity, C#, Visual Studio, Adobe Illustrator
