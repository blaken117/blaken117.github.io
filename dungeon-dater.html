Dungeon Dater

3D rogue-lite crossed with a dating sim

Check out the trailer here!

Capestone project spanning over 6 months. Collaborated with a large team taking on all responsiblities relating to programming and optimizations. Setup UX/ UI for the players HUD and game menus. Project covered all aspects of game creation including; ideation, documentation, development, play-testing, publishing and marketing. The aim of the game was to combine 2 games environments (rogue-lites and dating sims) into 1.

Contributions

  • Programmed game mechanics
  • Debugging and designer tools
  • Optimizations for room loading
  • Created UI elements via Adobe Illustrator & Photoshop
  • Setup basic enemy and boss AI
  • Implemented Ink Studios plugin for dialogue mechanics
  • Worked with mixamo animations for player movement
  • Basic shader and visual effects creation

    • Software Used

    Unity, C#, Visual Studio, Blender, Source Tree, Adobe Illustrator, Adobe Photoshop
