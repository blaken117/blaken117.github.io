Dog-go

Pet Simulator

Dog-go was an 8-week team project where I worked with another designer to create a mobile game on Andriod. This project aimed to develop a game that uses in-app purchases and analytics inside Unity. Dog-go was my first time making a mobile app, so it was great to learn touch controls for android and optimizing code for a mobile environment.

Contributions

  • Create serialization for the loading and saving of dog data
  • Learnt how to create Grass Shaders in C#
  • Scripted interfaces for mouse interactions using Unity
  • Setup mobile touch controls
