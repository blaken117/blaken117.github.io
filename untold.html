Through the Untold Paths

Autobiographical Game

This was a 3-week Uni project that required me to reflect on my life and choose a critical moment or aspect to convey/ communicate through game-play. I decided to focus on terms such as unknown, following and path, which lead to some unique mechanics. During this project, I learnt how to work with other peoples code via Unity plugins/ tools and used Unity's new input system to add controller support.

Contributions

  • Setup stencil shader for light masking
  • Learnt Unity's new input system for controller support
  • Created particle and text effects
  • Linked pathing tools to create basic AI
