Planet Ours

3D RTS Alien Simulation

This was a unique project where I learnt coding fundamentals such as Path Data and Pathfinding, AI and Behaviours, JSON serialization and how to use Unity's debugging tools. The aim was to produce a game where different actors would contribute to an overall goal and react to each other, creating unique behaviours/ interactions. I decided to make an RTS type game for my love of games such as Starcraft 2 and Age of Empires.

Contributions

  • Created 3D alien models via Blender
  • Learnt the basics of A* Pathfinding
  • Learnt how to structure path data code for Pathfinding
  • Learnt the basics of AI coding using the GOAP system
  • Scripted interfaces for unit actions such as click to move and unit selection
  • Created UI elements via Adobe Illustrator

    • Software Used

    Unity, C#, Visual Studio, Blender, Source Tree, Adobe Illustrator, Adobe Photoshop
