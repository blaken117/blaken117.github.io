DoorFare

3D Wave Clearer

DoorFare was a simply prototype I made when first learning to code. I had a 2 week window at the end of the semester to try put together a game from all the knowledge I had learnt about scripting. The aim was to create a game which had unique wave clearer mechanics and put my newly founded coding skills to the test.

Contributions

  • Setup Unity Nav Mesh for enemy Pathfinding
  • Learnt and Implemented Corountines for wave progression
  • Setup Unity colliders and triggers for enemy player detection
  • Created UI elements via Adobe Illustrator
